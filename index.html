04 — CAPSTONE
Automated Conveyor Filling & Capping Machine

Designed and implemented a fully automated conveyor belt system for liquid filling and bottle capping — integrating mechanical design, electronics, and embedded software. The system handles one bottle per cycle with high precision: sensors detect exact bottle position, a liquid dispensing unit delivers a predefined volume with minimal error, and a capping mechanism secures the lid automatically. An LCD screen provides real-time operator feedback. Results showed consistent bottle alignment, high filling accuracy, and reliable capping — a cost-effective solution for semi-automated production lines with potential for multi-bottle scalability.

Key Outcomes ✦ Precise liquid dispensing with minimal volumetric error
✦ Automated capping mechanism with consistent reliable force
✦ Real-time LCD monitoring interface for operators
✦ Scalable architecture ready for multi-bottle expansion
